# 04_non_goals.md

## 目的
本プロジェクトで**やらないこと**を明確にし、スコープ膨張を防ぎます。

## 非対象
- Web アプリ化
- スマホアプリ化
- クラウド同期
- マルチユーザー同時編集
- オンライン認証連携
- 外部 SaaS 依存前提の設計
- OCR 機能の初期搭載
- 多言語対応
- AI 推論機能の本体組込
- 複数 DB 製品対応
- ネットワーク共有環境での高頻度同時利用最適化

## 明示的に後回しのもの
- 高度な検索ランキング
- バルク編集 UI
- 大規模データ向けページング最適化
- 自動監査分析
- ダッシュボード高度可視化

## スコープ管理ルール
上記を実装したくなった場合は、必ず以下を行う。
1. `docs/97_open_issues_and_constraints.md` に記録
2. `docs/98_decision_log.md` で採否判断
3. 採用する場合は要件定義から更新する
