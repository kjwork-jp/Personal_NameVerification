# 03_glossary.md

## 用語集

### NameVerification
本プロジェクト全体、および最終アプリの名称。

### 名前
照合対象となる人名・表記名。  
本システムでは正規化後の一意性を重視する。

### タイトル
名前に紐づく作品・区分・大分類。  
例: 作品名、シリーズ名。

### サブタイトル
タイトル配下の小分類。  
例: 話数、章、エピソード、区分コード。

### リンク
名前とサブタイトルの関連付け。

### 正規化
表記ゆれを減らすための変換処理。  
全角半角、空白、大小文字などの差異を吸収する。

### 論理削除
データを物理削除せず、削除済み状態として扱うこと。

### 完全削除
物理的にレコードを除去すること。

### 監査ログ
誰が、いつ、何を変更したかを追跡するための記録。

### operator_id
操作主体を識別する ID。

### viewer
参照専用ロール。

### editor
作成・更新・レポート出力が可能なロール。

### admin
削除・復元・完全削除・インポート・復旧が可能なロール。

### UAT
User Acceptance Test。  
利用者観点で受入可否を判断するテスト。

### DoR
Definition of Ready。  
着手可能と判断する条件。

### DoD
Definition of Done。  
完了と判断する条件。
