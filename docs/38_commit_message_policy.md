# 38_commit_message_policy.md

## 形式
`<type>: <summary>`

## type
- docs
- feature
- fix
- refactor
- test
- chore

## 例
- `docs: add bootstrap governance documents`
- `feature: implement pyside6 main window`
- `fix: prevent viewer from destructive actions`

## 禁止
- `update`
- `fix`
- `wip`
- `misc`
だけの曖昧なメッセージ
