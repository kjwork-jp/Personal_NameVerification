# 18_search_behavior_spec.md

## 1. 目的
検索の挙動を固定し、実装差異を防ぐ。

## 2. 検索対象
- 名前
- タイトル
- サブタイトル名
- サブタイトルコード

## 3. モード
- 完全一致
- 部分一致

## 4. フィルタ
- タイトル
- 紐づき有無
- 削除済み含有有無
- ソート条件

## 5. 結果表示
- 一覧に最小情報を表示
- 詳細ペインに補足情報を表示
- 検索ヒット数を表示する

## 6. 入力挙動
- 前後空白は無視
- 正規化候補を内部処理に使う
- 空検索時の挙動は一覧表示にするか、設計決定に従う

## 7. 禁止事項
- あいまいな暗黙フィルタ
- 何がヒットしたか分からない結果表示
